# Oakville Home Seller Success

This is a Next.js web application for Ken Finch, a real estate agent specializing in the Oakville, Ontario market. The site is designed to attract and assist home sellers by providing valuable tools and a personalized experience.

## Features

*   **AI-Powered Home Valuation:** Potential sellers can get an instant, data-driven valuation of their home using a sophisticated AI tool.
*   **Secure Seller Portal:** Clients who sign up can access a private dashboard to track their home-selling journey.
    *   **Personalized Dashboard:** Displays the seller's latest AI valuation, upcoming appointments, and an interactive pre-listing checklist.
    *   **Document Hub:** A secure place for sellers to access all their important documents.
*   **Client Testimonials:** Showcases positive feedback from past clients to build trust and credibility.
*   **Seller & Admin Authentication:** Separate login flows for home sellers and for the site administrator to manage contacts.
*   **Contact Management:** An admin-only section to view contact form submissions from potential clients.

## Tech Stack

*   **Framework:** Next.js with React & TypeScript
*   **Styling:** Tailwind CSS with ShadCN UI components
*   **Backend & Database:** Firebase (Authentication, Firestore)
*   **Generative AI:** Google's Gemini models via Genkit
