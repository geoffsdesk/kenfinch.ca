# Oakville Home Seller Success: AI-Powered Real Estate Platform

Welcome to Oakville Home Seller Success, a modern web application built for Ken Finch, a top real estate agent in Oakville, Ontario. This platform is strategically designed to attract potential home sellers by offering an AI-powered home valuation tool and a secure, personalized portal for clients who are on their selling journey. The site also includes a private admin dashboard for lead and client management.

## Key Features

### For Home Sellers:

*   **Instant AI Home Valuation:** A sophisticated tool that allows potential sellers to receive a data-driven valuation of their home in real-time, based on a comprehensive set of property details and current market data.
*   **Secure Seller Portal:** Registered clients gain access to a private dashboard to manage and track every step of their home-selling process.
*   **Personalized Dashboard:** A central hub displaying the seller's latest AI valuation, a pre-listing preparation checklist, an AI assistant, and key dates.
*   **AI Seller Assistant:** A chatbot to coach sellers through the home preparation process, answer questions, and provide support.
*   **Document Hub:** A secure, centralized location for sellers to access all important documents, such as contracts, marketing materials, and closing papers.
*   **Client Testimonials:** Builds trust and credibility by showcasing positive feedback from past clients.

### For the Administrator (Ken Finch):

*   **Secure Admin Login:** A separate, secure login for the site administrator, accessible at `/login`.
*   **Contact Lead Management:** An interface to view and manage all contact form submissions from potential clients.
*   **Registered Seller Overview:** A comprehensive list of all registered sellers who have created an account.
*   **Valuation Insights:** The ability to view the detailed AI home valuations generated by each seller, providing critical data to facilitate informed and productive client conversations.
*   **Chatbot Log Review:** An admin-only page to review the conversation history between sellers and the AI assistant for quality assurance and insight into seller questions.

## Technology Stack

This project leverages a modern, robust tech stack for a high-performance, scalable, and feature-rich user experience:

*   **Framework:** [Next.js](https://nextjs.org/) with React & TypeScript (App Router)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) with [ShadCN UI](https://ui.shadcn.com/) for a beautiful and consistent component library.
*   **Backend & Database:** [Firebase](https://firebase.google.com/) for Authentication and Firestore DB.
*   **Generative AI:** [Google's Gemini models](https://deepmind.google.com/technologies/gemini/) via [Genkit](https://firebase.google.com/docs/genkit) for the AI home valuation feature.
*   **Hosting:** Deployed on [Firebase App Hosting](https://firebase.google.com/docs/hosting).
