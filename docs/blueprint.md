# **App Name**: Oakville Home Seller Success

## Core Features:

- Website: KenFinch.ca needs to be the front end of the application available on web and mobile to be the onboarding experience for home sellers.
- Seller Dashboard: Seller Dashboard: Track listing prep, photo shoots, and open house dates.
- Document Hub: Document Hub: Securely store staging guides, legal forms, and closing documents.
- Home Value Tool: AI-powered Home Value Prediction Tool: Provide a live valuation estimate that takes into account the local neighborhood trends in Oakville, and generates a confidence score. The confidence score reflects how closely the features of the home match recent sales data.

## Style Guidelines:

- Primary color: Soft Gold (#D4AF37) to communicate luxury and success, used for primary CTAs.
- Background color: Warm White (#F9F9F9) for a clean and modern base.
- Accent color: Muted Green (#A8C6A2) to reflect Oakville's natural beauty; used sparingly to draw attention to supporting information and wayfinding elements.
- Headers: 'Playfair', a traditional serif font for elegance. Note: currently only Google Fonts are supported.
- Body text: 'Inter', a clean, modern sans-serif for digital readability. Note: currently only Google Fonts are supported.
- Use simple, elegant icons to represent dashboard items and document types. Icons should have consistent line weights and be in muted green.
- Clean and intuitive layout with a focus on usability. Use of white space to ensure readability and scannability.
